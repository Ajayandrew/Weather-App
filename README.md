# 🌍 Weather Report Application

A responsive and innovative **Weather Report App** built with **React** and **Tailwind CSS**.  
This app allows users to search for any city and view **real-time weather data** with a modern, attractive UI.

## ✨ Features
- 🔎 **Search by City** – Enter any city name to fetch live weather data.
- 🌤 **Dynamic Weather Icons** – Icons change based on weather conditions (sun, rain, snow, etc.).
- 🎨 **Innovative UI** – Gradient backgrounds, glassmorphism cards, and smooth animations.
- 📱 **Responsive Design** – Works seamlessly across mobile, tablet, and desktop.
- 🔔 **Toast Notifications** – Instant feedback for success or error (invalid city).
- 🔢 **Badge Counter** – Tracks the number of searches made.
- 🌡 **Temperature Progress Bar** – Visual representation of temperature values.

---

## 🛠 Tech Stack
- **React** – Frontend framework
- **Tailwind CSS** – Styling and responsive design
- **Axios** – API requests
- **React Icons** – Weather icons
- **React Hot Toast** – Notifications
- **OpenWeatherMap API** – Weather data provider
